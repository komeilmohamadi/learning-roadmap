## Remote Repository
- `git remote add origin <url>`: connect local repo to GitHub
- `git remote -v`: list remote repositories

## Push & Pull
- `git push`: send local commits to remote repository
- `git pull`: fetch and merge remote changes into local branch

## git fetch vs git pull
- `git fetch`: downloads remote changes without merging
- `git pull`: downloads and automatically merges changes

Using fetch first helps review changes safely before merging.
